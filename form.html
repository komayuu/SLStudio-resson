属性値 入力内容
text テキスト
number 数値
password パスワード
email メールアドレス
radio ラジオボタンでの選択
checkbox チェックボックスでの選択
submit 送信ボタン
hidden ブラウザ上で表示せずに値を送信
属性名 用途
value 入力値を指定。type="submit"の場合はボタンに表示する文字列を指定。
name フォーム部品の名前を指定。サーバー側で値を受け取る際のキーになる。
checked チェックボックスやラジオボタンを選択。
required 入力必須。入力せずに送信しようとした場合、警告を表示。
placeholder 入力欄内の入力に関するヒントを示したテキストを指定。

属性名 用途
class 要素が属する種類や分類を示す名前
id 要素の名前、id(他のものと重複しない固有の名前)
lang 日本語や英語などの言語の種類
style CSSスタイル
hidden 要素を表示しない(※type="hidden"とは別物)
tabindex タブキーによる移動の順序、フォーカスの可/不可を制御
